Device Dimensions

Various mobile devices and screen sizes are used to access xelly.games.

It's a good idea to make your game responsive, allowing for a range of widths — widths between 340px to 412px are good targets. It's common, for example, to use engine.drawWidth and engine.drawHeight when laying out the entities of your game.

Aspect Ratio​

When you build your game, you'll target a given aspect ratio, one of:

  • default (4:3 ratio)
  • tall (≈ 4:5.2 — or 1.75x taller than the default ratio).

You can select the aspect ratio when you upload your game and when testing using the "utility" tool.
